# Student Data Cleaning Project

## Objective
Learn how to handle missing values in a dataset using pandas.

## Concepts Covered
- fillna()
- dropna()
- drop()

## Files
- students.csv
- student_cleaning.ipynb
- cleaned_students.csv

## Steps to Run
1. Open Jupyter Notebook in Anaconda
2. Open student_cleaning.ipynb
3. Run each cell in order
4. A cleaned file will be generated
